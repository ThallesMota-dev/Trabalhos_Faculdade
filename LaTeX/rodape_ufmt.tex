\section{Exemplo de Pseudocódigo}
Para ilustrar o uso em trabalhos de Algoritmos:

\begin{verbatim}
  função busca_binaria(vetor A, inteiro chave):
      esquerda ← 0
      direita  ← tamanho(A) - 1
      enquanto esquerda <= direita:
          meio ← (esquerda + direita) / 2
          se A[meio] == chave:
              retorne meio
          senão se A[meio] < chave:
              esquerda ← meio + 1
          senão:
              direita ← meio - 1
      retorne -1
\end{verbatim}
